# VEO3 v12 更新說明

## 更新日期
2024-12-23

## 新增功能

### 1. 生圖功能
- **圖片/視頻比例選擇**
  - 全視頻 (100%)
  - 全圖片 (100%)
  - 70% 視頻 + 30% 圖片
  - 50% 視頻 + 50% 圖片
  - 30% 視頻 + 70% 圖片
  - 自定義滑桿調整

- **8 種圖片生成模型**（按強度排序）
  1. Gemini 3 Pro Image (Nano Banana 2) - Google 最新
  2. GPT Image 1.5 - DALL-E 3 格式
  3. Midjourney - 藝術風格強
  4. Ideogram V3 - 文字渲染專家（中文最佳）
  5. Flux 1.1 Pro - 高質量
  6. Flux Schnell - 最快速
  7. Stable Diffusion XL - 開源靈活
  8. 豆包圖片 - 中國本土

- **備用模型鏈**
  - 自動從最強模型切換到備用模型
  - 當 API 失敗時自動切換

- **圖片時長設定**
  - 2 秒/張（4 張/段）
  - 3 秒/張（3 張/段）
  - 4 秒/張（2 張/段）

- **13 個 API Key 輪流**
  - 圖片生成也使用輪流機制
  - 避免限流

### 2. 外部上傳功能
- **上傳視頻** - 每個片段都可上傳外部視頻
- **上傳圖片** - 每個片段都可上傳多張圖片
- **上傳音頻** - 每個片段都可上傳外部音頻
- 適用於 API 生成失敗時的備用方案

### 3. 字幕功能
- **字幕模式**
  - AI 自動識別
  - 手動編輯
  - 無字幕

- **字體選擇**（6 種）
  - 思源黑體、思源宋體、粉圓體
  - 霄宮文楷、蘋方黑體、微軟正黑體

- **字體大小**（4 種）
  - 小、中、大、特大

- **字體顏色**（6 種）
  - 白色、黃色、青色、綠色、粉色、橙色

- **字框樣式**（6 種）
  - 無字框、陰影、描邊
  - 黑底字框、藍底字框、漸變字框

- **字幕位置**（5 種）
  - 底部居中、底部左側、底部右側
  - 頂部居中、中間居中

- **預覽功能**
  - 可即時預覽字幕效果

### 4. 所有模式適用
以上功能適用於所有 4 個模式：
- 快速模式
- 高質量模式
- 固定人物模式
- 劇情模式

## 技術改進
- 圖片生成使用 13 個 API Key 輪流
- 備用模型自動切換機制
- 片段預覽組件支持圖片上傳
