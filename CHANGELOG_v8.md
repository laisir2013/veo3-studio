# VEO3 v8 版本更新說明

## 更新日期
2025-12-22

## 主要更新

### 片段預覽佈局重構
將原本的 4 列網格佈局改為**一行一框一影片**的列表式設計。

#### 新增組件
- `client/src/components/SegmentListPreview.tsx` - 新的列表式片段預覽組件

#### 新佈局特點

1. **一行一框一影片**
   - 每個片段獨立佔據一整行
   - 更清晰的視覺層次

2. **清晰的狀態顯示**
   - 等待中：灰色邊框
   - 生成中：藍色邊框 + 旋轉圖標
   - 已完成：綠色邊框 + 勾選圖標
   - **失敗：紅色邊框 + 錯誤圖標**（非常醒目）

3. **信息展示**
   - 左側：片段編號 + 視頻縮略圖
   - 中間：時間範圍 + 狀態 + 場景描述
   - 右側：批次標籤 + 操作按鈕

4. **可展開詳情**
   - 點擊展開按鈕查看大型視頻預覽
   - 顯示場景描述、旁白文字、生成提示詞
   - 失敗時顯示詳細錯誤信息

5. **統計欄**
   - 頂部顯示總計片段數
   - 分類顯示：已完成、生成中、失敗、等待中
